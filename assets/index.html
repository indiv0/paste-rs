paste-rs

A paste tool written entirely in rust

I've also written a command-line tool in Python3. Input your paste via stdin and a URL to the paste (in text/plain) is returned on stdout. The script is available here or you can install it by running:

curl p.nikitapek.in/script > pasters; chmod 755 pasters; su -c "mv pasters /usr/local/bin"

to download it and put it in your path (do you trust me :) ?).

This entire system is written in rust. For more information (and the source), check out this project's GitHub page here.
