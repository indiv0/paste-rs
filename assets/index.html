Paste


I've also written a command-line tool in Python3. The paste is input via stdin and an url to the paste (in text/plain) is returned on stdout. The script is available here or you can install it by running:

        curl p.nikitapek.in/script > paste; chmod 755 paste; su -c "mv paste /usr/local/bin"

to download it and put it in your path (do you trust me :) ?).


This entire system is written in rust. For more information (and the source), check out this project's GitHub page here.
